# Payment Link Search and Filter Guide

This guide explains how to display and filter payment links within the Ottu Dashboard. By following the steps below, you will be able to view payment links created by a specific user or processed through a specific payment gateway.

## [Step-by-Step Instructions](payment-link-search-and-filter-guide.md#step-by-step-instructions)

{% stepper %}
{% step %}
Access the Transactions Section

From the Ottu Dashboard, open the **Payment Request** plugin and go to the **Transactions** subsection.\
Here, all transactions performed via the Payment Request plugin are listed.
{% endstep %}

{% step %}
Add Payment Link Headers

To display payment links and gateway names in the transactions table:

* Navigate to the Table Headers subsection under **Settings**.
* From the available headers, search for **Payment URL** and add it to the selected headers.
* Also, add **Gateway Name** to the selected headers.

When you return to the **Transactions** table, the **Payment URL** and **Gateway Name** columns will be visible.
{% endstep %}

{% step %}
Filter by User

* Click on the **Filter** option.
* In the **Created By** field, enter the username of the user.
* Click **Search**.

All payment links created by the specified user will be displayed.
{% endstep %}

{% step %}
Filter by Payment Gateway

* In the filter options, specify the desired **Payment Gateway**.
* Click **Search**.

All payment links associated with the selected payment gateway will be displayed.
{% endstep %}
{% endstepper %}
